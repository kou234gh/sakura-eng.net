なぜここに作るか。
XSERVERはこのpublic_html直下のディレクトリをサブドメインとして認識するから。同ドメイン内である理由は、好奇心・見栄えがいいから。
エックスサーバーがポートを制限しており、同ドメイン別ディレクトリにサーバーを立てることができても、htaccessを通らなければportを移動できず、POSTによるバックエンド処理を行えないため。GmailAPIの使用を断念。
